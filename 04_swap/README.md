# 課題04：メモリ余裕あるのにサーバーがめちゃくちゃ遅い

## 状況

- アプリサーバー。昼間は普通に動いているが、夜間バッチ実行後から重くなってきた
- `free` を確認したら「まだメモリ余裕あるじゃん」と言われた
- でも操作がもっさりしていて、SSHすら繋がりにくい

## やること

1. `logs/` の中のファイルをAIに全部渡す
2. 「なぜサーバーが遅いのか診断してください」と聞く
3. 答え合わせは `answer.md`

## 渡すファイル

- `free.txt` — メモリ使用状況
- `vmstat.txt` — メモリ/swap統計
- `top.txt` — プロセス一覧と全体リソース
- `iostat.txt` — ディスクI/O詳細
