# 課題05：リソースは全部正常なのに突然繋がらなくなった

## 状況

- Webアプリサーバー（外部APIを頻繁に呼び出す構成）
- 突然「接続できない」エラーが多発し始めた
- CPU・メモリ・ディスクを確認したが全部正常。アプリも動いている
- 「何も変えてないのになぜ？」という状況

## やること

1. `logs/` の中のファイルをAIに全部渡す
2. 「なぜ接続エラーが発生しているのか診断してください」と聞く
3. 答え合わせは `answer.md`

## 渡すファイル

- `top.txt` — プロセス一覧と全体リソース
- `free.txt` — メモリ使用状況
- `df_h.txt` — ディスク使用量
- `netstat.txt` — TCP接続状態の一覧（抜粋）
- `ss_summary.txt` — ソケット統計サマリー
