# 課題03：ディスク残量あるのにファイルが作れない

## 状況

- ログ収集サーバー。アプリのログを `/var/log/app/` に書き込んでいる
- 突然アプリからエラーが出始めた：「No space left on device」
- でも担当者が `df -h` を確認したら「全部余裕ある。なんで？」と混乱している

## やること

1. `logs/` の中のファイルをAIに全部渡す
2. 「なぜ 'No space left on device' が出るのか診断してください」と聞く
3. 答え合わせは `answer.md`

## 渡すファイル

- `df_h.txt` — ディスク使用量（容量ベース）
- `df_i.txt` — ディスク使用量（inodeベース）
- `top.txt` — プロセス一覧と全体リソース
- `free.txt` — メモリ使用状況
