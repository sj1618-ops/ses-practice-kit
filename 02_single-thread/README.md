# 課題02：CPU余裕あるのになんで遅いの？

## 状況

- バッチ処理サーバー。定期的に集計処理を走らせている
- 担当者が「topを見たらCPU 25%で余裕あるのに、処理が全然終わらない」と困っている
- 「サーバーのスペック上げれば解決しますか？」と聞いてきた

## やること

1. `logs/` の中のファイルをAIに全部渡す
2. 「なぜ処理が遅いのか、スペックアップで解決するか診断してください」と聞く
3. 答え合わせは `answer.md`

## 渡すファイル

- `top.txt` — プロセス一覧と全体リソース
- `mpstat.txt` — CPUコアごとの使用率
- `vmstat.txt` — システム全体の統計
- `free.txt` — メモリ使用状況
