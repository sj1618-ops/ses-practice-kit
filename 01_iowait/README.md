# 課題01：なんかサーバーが重いんですけど

## 状況

- 本番のWebアプリ（ECサイト）が朝から重い
- 「CPU張り付いてます」と報告が上がってきた
- `logs/` の中に現場から送られてきたログがあります

## やること

1. `logs/` の中のファイルをAIに全部渡す
2. 「何が原因ですか？」と聞く
3. AIの診断を読む
4. 答え合わせは `answer.md`

## 渡すファイル

- `top.txt` — プロセス一覧と全体リソース
- `vmstat.txt` — CPU/メモリ/IO統計
- `iostat.txt` — ディスクI/O詳細
- `free.txt` — メモリ使用状況
